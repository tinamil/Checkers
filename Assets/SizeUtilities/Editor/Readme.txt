How to use:
*Download and import this package.
*Window>Size Utilities to display the new view (hint: dock the new viewbelow the Inspector view)

Active object size is displayed in x,y and z fields (aligned with local coordinates system). Units are Unity units.
Size fields are read and write enabled: type in a new size and local scale will automatically be computed.
Click on the Set as unit button to set localScale to (1,1,1) without changing the object size.
